# Weather Dashboard App 

A simple React weather application that allows you to search for any city and view current weather, wind, humidity, and other atmospheric data.

## Features

- Search for any city and get real-time weather data.
- Displays temperature in Celsius or Fahrenheit.
- Shows wind speed, direction, humidity, visibility, precipitation, and UV index.
- Responsive design with modern cards layout.
- Clean and minimal UI.


## Technologies Used

- React.js
- Custom Hooks (`useWeather`)
- CSS for styling
- Weather API ((https://www.weatherapi.com/))
